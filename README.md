## 效果示例

> 用户 › 我想做副业，我是运营，小红书有14w粉丝但两年没运营了，不想靠接广告
>
> 副业顾问 › 你的14w粉丝是被严重低估的资产。即使数据下滑，它是任何新入局者花几十万广告费都买不来的信任背书。
> 结合你的UGC趋势洞察能力，首选方向是：小红书账号运营顾问（1对1咨询+月度陪跑）。
> 定价599元/次诊断，3000元/月陪跑，稳定6个客户后月收入可达18,000-36,000元。
> 最快路径：今天发一条「我在接账号诊断，限5个名额」，最快48小时内收到第一笔钱。

---

> 用户 › 我是前端工程师，每周只有5小时，想尽快有收入
>
> 副业顾问 › 5小时/周，最快变现的路是接外包单子。

## 覆盖的副业方向

| 方向 | 适合人群 | 最快见收入 | 月收入上限 |
|------|---------|-----------|----------|
| 知识付费（1对1咨询） | 有专业技能的任何人 | 1-2周 | 50,000+ |
| 自由职业接单 | 设计、研发、文案 | 2-4周 | 40,000+ |
| 账号运营顾问 | 运营、市场 | 1-2周 | 36,000+ |
| AI工具服务 | 产品、研发、运营 | 2-4周 | 25,000+ |
| 内容创作账号 | 运营、市场、产品 | 3-6月 | 无上限 |
| 独立产品（SaaS/插件） | 研发、产品 | 3-12月 | 无上限 |

## 工作流程

Skill 会逐步提问 7 个关键问题，不一次性轰炸：

```
职业方向 → 技能亮点 → 可用时间 → 收入目标
→ 见收入速度 → 风险偏好 → 已有资源
```

收集完成后输出四段式报告：

```
① 核心优势诊断
② 推荐副业方向（首选+备选）
③ 30天MVP启动计划（Week by Week）
④ 收入预估表（保守/乐观双版本）
```

## 安装方法

**方式一：git clone**

```bash
git clone https://github.com/mayaoting20-sys/fuye-advisor.git ~/.claude/skills/fuye-advisor
```

**方式二：下载 zip**

下载本仓库 zip，解压到 `~/.claude/skills/fuye-advisor/`

安装后，在 Claude Code 中说「我想做副业」即可触发。

## 诚实边界

- 收入数据基于市场调研区间，个人实际结果差异较大
- Skill 给方向和计划，执行还是要靠你自己
- 不预测黑天鹅，不保证一定成功
- 数据截至 2025-2026 年中国市场，海外情况请自行参考

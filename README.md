# 副业顾问 Fuye Advisor

> 帮助互联网从业者找到最适合自己的副业，并在30天内完成第一个可验证的MVP。

## 这个 Skill 能做什么

输入你的情况，输出一份完整的副业规划：

- **精准匹配** — 根据你的职业背景、可用时间、收入目标，从6大副业方向里找最适合你的
- **30天MVP计划** — 不是泛泛而谈，是今天就能开始执行的 Week by Week 行动清单
- **收入预估** — 保守/乐观双版本，附带关键前提条件，不画大饼
- **0成本启动** — 每个方向都提供不花钱就能验证的第一步

## 覆盖的副业方向

| 方向 | 适合人群 | 最快见收入 |
|------|---------|-----------|
| 知识付费（1对1咨询） | 有专业技能的任何人 | 1-2周 |
| 自由职业接单 | 设计、研发、文案 | 2-4周 |
| 小红书/抖音账号运营顾问 | 运营、市场 | 1-2周 |
| 独立产品（SaaS/插件） | 研发、产品 | 3-12月 |
| AI工具服务 | 产品、研发、运营 | 2-4周 |
| 内容创作账号 | 运营、市场、产品 | 3-6月 |

## 使用方式

安装到 Claude Code 后，触发词：

```
我想做副业
帮我找副业
适合我的副业是什么
互联网副业怎么做
```

Skill 会逐步提问7个关键问题，然后输出：诊断报告 → 推荐方向 → 30天计划 → 收入预估。

## 安装方法

### 方式一：直接下载文件夹

```bash
git clone https://github.com/mayaoting20-sys/fuye-advisor.git ~/.claude/skills/fuye-advisor
```

### 方式二：下载 zip 包

下载 `fuye-advisor.zip`，解压到 `~/.claude/skills/fuye-advisor/`

## 文件结构

```
fuye-advisor/
├── SKILL.md                        # 主文件，定义4阶段工作流
└── references/
    ├── hustle-matrix.md            # 职业×副业匹配矩阵
    ├── mvp-playbooks.md            # 6大副业30天MVP方案
    └── income-benchmarks.md        # 收入区间&时间线基准数据
```

## 适合谁用

- 想要快速验证而不是空想
